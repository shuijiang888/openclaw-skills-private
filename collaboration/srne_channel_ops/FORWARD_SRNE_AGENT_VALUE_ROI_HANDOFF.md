## 三、评估公式（定稿定义）

记（单位均为 **人民币 · 万元**，年）：

| 符号 | 含义 |
|------|------|
| \(H\) | 渠道运营相关周工时（团队合计，小时/周） |
| \(c\) | 综合人力成本（万元/小时） |
| \(e\%\) | 上线后工时节省比例 |
| \(R\) | 风险与机会价值（演示口径，万元/年） |
| \(O\) | 其他硬节省（万元/年） |
| \(S\) | 系统年费（万元/年） |

1. **年工时成本（现状）** \(L_0 = H \times 52 \times c\)。
2. **年工时成本（上线后）** \(L_1 = L_0 \times (1 - e/100)\)。
3. **工时节省额** \(\Delta L = L_0 - L_1 = L_0 \times e/100\)。
4. **年总收益（演示）** \(B = \Delta L + R + O\)。
5. **年净收益** \(N = B - S\)。
6. **ROI（倍）** \(= N / S\)（仅当 \(S>0\)）。
7. **静态回收期（月）** \(= S / (N/12)\)（仅当 \(N>0\) 且 \(S>0\)）。

**表格语义（Cursor 修订点）：**  
「年总收益（演示合计）」行仅在中栏用文字说明构成，**避免**将「工时上线后成本」与「风险金额」错误相加到同一「上线后」数字格，防止量纲误读。

## 四、预填默认值评估（是否合理）

| 参数 | 默认值 | 评估 |
|------|--------|------|
| 系统年费 | 14 万/年 | 中小 SaaS/模块年费数量级，可调低或调高对齐商务报价。 |
| 周工时 | 26 h | 约 0.6～0.7 FTE 花在渠道运营相关事务上，偏**中等体量**团队，合理。 |
| 节省比例 | 33% | 作战台+情报+导入+预警闭环的**保守～中性**假设，可演示；若组织极成熟可调低。 |
| 万元/小时 | 0.042 | 约 **336 元/小时**全负荷成本，与一线+公摊混合口径一致，合理。 |
| 风险与机会 | 14 万/年 | **演示占位**，对应回款/预警/丢单避免等，需售前按客户改。 |
| 其他硬节省 | 5 万/年 | 会务差旅等，演示向。 |

**默认下数量级自检：**  
\(L_0 \approx 26×52×0.042 ≈ 56.8\) 万；\(\Delta L \approx 18.7\) 万；\(B \approx 37.7\) 万；\(N \approx 23.7\) 万；**ROI ≈ 1.7×**，回收期约 **7 个月** — 适合向管理层展示「方向为正、非天文数字」的演示，**不承诺**真实项目回报。

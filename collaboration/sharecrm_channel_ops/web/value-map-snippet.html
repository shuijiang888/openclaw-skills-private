◇ 各角色 · 价值一句话

同一套数据，不同角色看到不同决策切片；权限模型已演示（销售仅看自己负责渠道）。

高管 / 老板

「钱从哪来、风险在哪、这周必须盯谁」

  • 运营总览 + 业务作战台：目标脉搏、预警 SLA、优先动作清单
  • 绩效看板 BSC：财务 / 客户 / 流程 / 学习成长四象限对齐汇报口径
  • 高管简报：一键复制文本，会后直发群

区域总监 / 运营负责人

「区域打法、资源投向、团队复盘」

  • 区域出货、分级结构、TOP 渠道、负责人战报（含未分配渠道）
  • 预警闭环 + 渠道详情 业务上下文（情报 × 预警 × 国家）
  • 数据导入 校验预览 + 批次审计

一线销售 / 渠道经理

「出门带什么、回来记什么」

  • 所辖渠道列表、详情、月度补录、备注协作
  • 市场情报 + 规则型报价：拜访前热身与现场算价演示
  • 预警确认、库龄可见，避免「不知道催谁」

IT / 数字化（共创方）

「能接什么、边界在哪」

  • REST API + SQLite 演示栈；可替换为 PG / 中台同步
  • JWT、数据范围、导入审计已可讲清安全与合规故事
  • 下阶段对接 ERP/CRM、HTTPS 生产加固、只读演示模式等可签技术附件

阶段 A · 演示与共识（当前）

成交物：原型验收 / 需求确认纪要

高保真演示环境、种子数据、API 清单、部署手册；用于老板与业务统一语言。

阶段 B · 试点上线（3–8 周级）

成交物：试点租户 + 对接计划

  • 生产 HTTPS、密钥托管、CORS 与访问域收紧
  • 目标值从「演示口径」→ ERP/董事会下达口径同步
  • CSV RFC4180、错误行导出、操作时间轴（轻量审计）
  • 可选：只读演示模式、水印、字段脱敏规则

阶段 C · 规模推广（持续）

成交物：区域推广包 + 集成里程碑

  • 对接 CRM / 工单 / 财务应收（主数据与预警回写）
  • PDF/Excel 经营简报、订阅推送（企微/邮件）
  • 多币种、多法人、更细角色与字段级权限
  • 沙盘模拟「假设多签 N 家」对区域 BSC 的影响（高管演示增强）

◎ 分场景售卖 · 讲标与增购话术抓手

周复盘

董事会 / 月度经营会

区域扩张
